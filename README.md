# voice_controlled_object_detection
# Приложение для поиска объекта в видеопотоке по голосовой команде

Разработано для конкурса "Инженеры будущего" (компетенция "Искусственный интеллект")

## Описание проекта
Приложение позволяет в реальном времени находить заданный объект в видеопотоке с веб-камеры. 
Пользователь может голосом указать, какой объект искать (например, "найди книгу" или "где ноутбук?"), 
и нейронная сеть выделит найденный объект рамкой.

## Поддерживаемые классы объектов
- Шариковая ручка (ballpoint pen)
- Мобильный телефон (mobile phone)
- Ноутбук (laptop)
- Мячик для пинг-понга (ping-pong ball)
- Человек (person)
- Книга, записная книжка (book, notebook)
- Сумка, рюкзак (bag, backpack)

## Структура репозитория
| Файл/папка | Назначение |
|------------|------------|
| `main_app_objects.py` | Основное приложение с графическим интерфейсом (PyQt5) |
| `train_model.py` | Скрипт для дообучения модели нейронной сети |
| `voice_control.py` | Модуль распознавания голосовых команд (Vosk) |
| `plot_metrics_objects.py` | Построение графиков метрик, ROC-кривых, confusion matrix |
| `[твоя_фамилия]_best_model.pt` | Лучшая сохраненная модель (результат обучения) |
| `data.yaml` | Конфигурация данных для обучения YOLO |
| `requirements.txt` | Список необходимых библиотек |
| `vosk-model-small-ru-0.22/` | Модель для распознавания русской речи |
